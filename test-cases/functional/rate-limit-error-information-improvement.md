# Улучшение информации об ошибках рейт-лимитов

## Описание

Предложение по улучшению сообщений об ошибках, связанных с ограничениями количества запросов (rate limit).

Текущая информация недостаточно подробно объясняет пользователю причину возникновения ошибки и возможные способы решения проблемы.

---

## Шаги воспроизведения

1. Открыть сервис.
2. Выполнить действия, приводящие к срабатыванию ограничения рейт-лимита.
3. Открыть подробную информацию об ошибке.

---

## Текущий результат

Экран с информацией об ошибке, связанной с рейт-лимитами, содержит ограниченное описание проблемы.

Пользователю недостаточно информации для понимания:

- почему возникла ошибка;
- когда ограничение будет снято;
- какие действия можно выполнить для решения проблемы.

---

## Предлагаемый результат

Сообщение об ошибке должно содержать более подробную информацию:

- причину возникновения ограничения;
- информацию о текущем лимите;
- время ожидания до восстановления доступа (если возможно);
- рекомендации по дальнейшим действиям пользователя.

Пример:

> Вы превысили максимальное количество запросов. Попробуйте повторить действие через несколько минут.
