# HTML-сущность кавычек нарушает отображение содержимого отчёта

## Описание

Обнаружен дефект обработки пользовательского ввода: при использовании HTML-сущности кавычек (`&quot;`) в поле **«Фактический результат»** или **«Ожидаемый результат»** после сохранения отчёта его содержимое отображается некорректно. При повторном открытии отчёта отображается только заголовок, а остальная информация отсутствует.

---

## Шаги воспроизведения

1. Начать создание нового отчёта.
2. В поле **«Фактический результат»** или **«Ожидаемый результат»** ввести HTML-сущность кавычек:

```html
&quot;
```

3. Сохранить отчёт и закрыть его.
4. Повторно открыть сохранённый отчёт.

---

## Фактический результат

После повторного открытия отчёта отображается только его заголовок. Остальное содержимое отчёта не отображается.

---

## Ожидаемый результат

При использовании HTML-сущности кавычек (`&quot;`) отчёт должен корректно отображать всё содержимое. Введённый текст должен сохраняться и отображаться без потери данных или нарушения структуры страницы.
