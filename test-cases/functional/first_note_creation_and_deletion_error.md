# Ошибка при создании и удалении первой заметки при отсутствии других заметок

## Описание

Обнаружен дефект при работе с первой заметкой пользователя. Если у пользователя отсутствуют заметки, при создании первой заметки возникают ошибки в консоли. После её удаления приложение может перейти в состояние бесконечной загрузки и отображать сообщение **«Ошибка доступа»**.

---

## Шаги воспроизведения

1. Удалить все существующие заметки (если они есть).
2. Создать новую заметку.
3. Удалить созданную заметку.

> **Примечание:** если ошибка не воспроизводится с первого раза, повторить шаги 2–3. После обновления страницы вероятность воспроизведения увеличивается.

---

## Фактический результат

- При создании первой заметки в консоли браузера появляется ошибка:

```text
Uncaught ReferenceError
```

- После удаления заметки отображается бесконечный индикатор загрузки и сообщение **«Ошибка доступа»**.
- В консоли браузера также появляется ошибка:

```text
Uncaught (in promise)
```

---

## Ожидаемый результат

- Первая заметка должна создаваться без ошибок в консоли браузера.
- После удаления единственной заметки приложение должно корректно обработать отсутствие заметок.
- Не должны появляться бесконечная загрузка, сообщение **«Ошибка доступа»** или ошибки в консоли.
