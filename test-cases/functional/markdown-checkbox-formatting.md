# Ошибка отображения Markdown чекбоксов в заметках

## Описание

Обнаружена ошибка при отображении чекбоксов в Markdown-разметке заметок.

При использовании синтаксиса Markdown для чекбоксов элементы отображаются как обычный список вместо интерактивных отмеченных и неотмеченных пунктов.

---

## Шаги воспроизведения

1. Открыть сервис.
2. Создать или открыть заметку.
3. Добавить Markdown-разметку чекбокса:

```markdown
- [x] Отмеченный пункт
- [ ] Неотмеченный пункт
```

4. Сохранить заметку.

---

## Фактический результат

Чекбоксы не отображаются в виде отмеченных и неотмеченных элементов.

Вместо этого содержимое отображается как обычный маркированный список:

```text
- [x] Отмеченный пункт
- [ ] Неотмеченный пункт
```

---

## Ожидаемый результат

Markdown-чекбоксы должны корректно отображаться:

- отмеченные пункты должны иметь состояние checked;
- неотмеченные пункты должны иметь состояние unchecked;
- пользователь должен визуально различать статус каждого пункта.

Пример ожидаемого отображения:

☑ Отмеченный пункт  
☐ Неотмеченный пункт
