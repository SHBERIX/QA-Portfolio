# Сообщение об ошибке валидации не исчезает после ввода корректной ссылки

## Описание

Обнаружен дефект валидации поля ссылки при изменении параметров общего доступа к набору. После ввода некорректного значения отображается сообщение об ошибке, однако после исправления ссылки на корректную сообщение не исчезает.

---

## Шаги воспроизведения

1. Открыть любой набор.
2. Нажать кнопку **«Поделиться»**.
3. Ввести ссылку в некорректном формате.
4. Исправить значение, введя ссылку в корректном формате.

---

## Фактический результат

После ввода корректной ссылки сообщение об ошибке **«Разрешены только английские символы и цифры»** продолжает отображаться, несмотря на то что введённое значение соответствует требованиям.

---

## Ожидаемый результат

После ввода ссылки в корректном формате сообщение об ошибке должно автоматически исчезнуть. Поле должно считаться валидным, а пользователь должен понимать, что ссылка успешно прошла проверку.
