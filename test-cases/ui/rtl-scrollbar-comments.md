# Проблема с отображением скроллбара при смене языка на Persian (فارسی) в окне фотографии

## Шаги воспроизведения

1. Поменять язык интерфейса на **"فارسی" (Persian)** или другой язык с направлением письма справа налево.
2. Открыть фотографию, у которой имеются комментарии.

---

## Фактический результат

При смене языка на **"فارسی" (Persian)** и открытии фотографии с комментариями скроллбар (полоса прокрутки) перекрывает кнопку **"Лайк"** и количество лайков у комментария.

Из-за этого данные элементы становятся недоступными для пользователей.

---

## Ожидаемый результат

При смене языка на **"فارسی" (Persian)** и открытии фотографии скроллбар не должен перекрывать кнопку **"Лайк"** и количество лайков у комментария.

Отображение элементов должно быть корректным и доступным для пользователей, чтобы они могли взаимодействовать с кнопкой лайка и видеть количество лайков без перекрытий или препятствий.
