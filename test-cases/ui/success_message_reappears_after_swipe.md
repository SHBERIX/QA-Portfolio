# Сообщение «Запись добавлена» повторно отображается после свайпа

## Описание

Обнаружен дефект отображения уведомления об успешном создании заметки. После смены типа хранения и создания новой заметки сообщение **«Запись добавлена»** можно скрыть свайпом, однако при повторном свайпе по кнопке **«Добавить»** оно снова появляется на короткое время и затем исчезает.

---

## Шаги воспроизведения

1. Открыть сервис и перейти на экран создания новой заметки.
2. Ввести текст заметки и сохранить её.
3. Изменить тип хранения (например, с **локального хранилища** на **сервер**).
4. Создать новую заметку.
5. Свайпнуть вправо, чтобы скрыть сообщение **«Запись добавлена»**.
6. Повторно выполнить свайп по кнопке **«Добавить»**.

---

## Фактический результат

После скрытия сообщение **«Запись добавлена»** временно исчезает, однако при повторном свайпе по кнопке **«Добавить»** снова появляется на короткое время, после чего повторно скрывается.

---

## Ожидаемый результат

После скрытия сообщение **«Запись добавлена»** должно оставаться скрытым и не появляться повторно при последующих свайпах или других действиях пользователя. Сообщение должно отображаться только после следующего успешного создания заметки.
