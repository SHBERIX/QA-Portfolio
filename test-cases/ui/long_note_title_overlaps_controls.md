# Длинный заголовок заметки перекрывает элементы управления

## Описание

Обнаружен дефект пользовательского интерфейса: при создании заметки с длинным заголовком текст выходит за пределы контейнера и перекрывает элементы управления, ухудшая читаемость и взаимодействие с интерфейсом.

---

## Шаги воспроизведения

1. Открыть сервис.
2. Создать новую заметку.
3. Ввести длинный заголовок (более 100 символов).
4. Сохранить заметку.
5. Открыть список заметок.

---

## Фактический результат

Длинный заголовок выходит за границы карточки заметки и перекрывает кнопку меню или дату создания заметки. Интерфейс отображается некорректно.

---

## Ожидаемый результат

Длинный заголовок должен корректно отображаться в пределах карточки заметки.

Ожидаемое поведение:

- текст переносится на новую строку или обрезается многоточием (`...`);
- элементы управления не перекрываются;
- структура карточки остается неизменной.
