# Треки слайдеров не отображаются в светлой теме

## Описание

Обнаружен дефект интерфейса: после переключения приложения на **светлую тему** треки слайдеров в настройках перестают отображаться. В результате пользователю сложно определить диапазон и текущее положение ползунка.

---

## Шаги воспроизведения

1. Запустить приложение.
2. Перейти в раздел **Настройки**.
3. Включить **светлую тему**.

---

## Фактический результат

Треки, по которым перемещаются ползунки слайдеров, не отображаются в светлой теме.

---

## Ожидаемый результат

Треки слайдеров должны корректно отображаться в светлой теме, обеспечивая удобное взаимодействие с элементами управления.
